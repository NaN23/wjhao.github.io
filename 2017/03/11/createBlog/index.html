2017-03-11 • 分类: 技术 • 标签: Hexo

起因&面对对象

配置hexo

hexo里最为常用的2个配置文件一个为hexo_config.yml，另一个为hexo\themes\xxx_config.yml（xxx为你使用的主题名字，默认为landscape）。关于主题，入门者可以在网络上搜到很多hexo主题，可以挑选自己喜欢的，有能力者可以自己二次修改，或者自己创建一个主题。
修改主题的方式：将主题clone到本地后，将主题的文件夹放置到…\hexo\themes中，将…\hexo_config.yml中theme后面的名字改为你的主题文件夹名字即可。

首先，在Git Bash中执行命令：hexo new “my new post”
其次，在…\hexo\source_post中打开my-new-post.md，打开方式可以选择记事本。因为使用的是Markdown，也可以下载专用的markdown编辑器。
